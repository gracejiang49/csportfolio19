Intro to Programming

Introduction to Programming is a UC approved course from 10th to 12th graders. This course focuses on learning HTML, CSS, and Javascript for web development. Various skills are practiced in the process of completing project work, such as problem-solving, self-motivation, explaining complex concepts, and vulnerability. Students spend part of their course time learning and practicing new knowledge in checkpoints where teachers can provide structured feedback. Once students are prepared, they apply these skills and knowledge to constructing webpages, like this one!
